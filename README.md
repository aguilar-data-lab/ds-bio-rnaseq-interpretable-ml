# End-to-End RNA-seq Analysis with Interpretable ML

## Overview
This project demonstrates an end-to-end RNA-seq analysis pipeline, from raw sequencing data to biologically interpretable machine learning insights

The emphasis is on:
- reproducible data processing
- principled model selection
- interpretability over black-box performance
- clear biological reasoning
